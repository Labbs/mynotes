# Zotion

Small application for have a notion like

## Development

### Requirements
The API is developed with Golang
The libraries are:
- fiber v2 (web framework)
- gorm (database access)
- zerolog (logger)
- goose (migration)

The UI is developed with Vue 3
The libraries are:
- Vue 3
- TipTap (WYSIWYG)
- Tailwind CSS

### Informations
The API is compatible with SQLite and Postgresql.
